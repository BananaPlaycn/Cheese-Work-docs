Gallery
=======

PyQt-Fluent-Widgets provides a gallery of examples to demonstrate its capabilities.

Running the gallery
-------------------

You can run the gallery example to see all the components in action:

.. code-block:: bash

    cd examples/gallery
    python demo.py

Components
----------

The gallery includes the following components:

- Buttons
- Inputs
- Dialogs
- Menus
- Toolbars
- Status bars
- Tabs
- Lists
- Tables
- Trees
- Progress bars
- Sliders
- Switch buttons
- Radio buttons
- Check boxes
- Labels
- Text edits
- Combo boxes
- Date/time pickers
- Color pickers
- File dialogs
- Message boxes
- Toast notifications
- Tool tips
- Status tips
