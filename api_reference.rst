cheese work
=============

This section provides detailed API documentation for PyQt-Fluent-Widgets.

Core components
---------------

- `FluentWindow` - Main window with navigation bar
- `NavigationBar` - Navigation bar component
- `NavigationPanel` - Navigation panel component
- `SettingsCard` - Settings card component
- `SettingsGroup` - Settings group component
- `SettingsPanel` - Settings panel component

Buttons
-------

- `PrimaryPushButton` - Primary style button
- `AccentPushButton` - Accent style button
- `TextPushButton` - Text style button
- `ToolButton` - Tool button
- `ToggleButton` - Toggle button
- `SwitchButton` - Switch button

Inputs
------

- `LineEdit` - Text input field
- `TextEdit` - Multi-line text input field
- `SpinBox` - Number input field
- `DoubleSpinBox` - Floating point number input field
- `ComboBox` - Drop-down list
- `CheckBox` - Check box
- `RadioButton` - Radio button

Dialogs
-------

- `Dialog` - Base dialog class
- `MessageBox` - Message box
- `FileDialog` - File dialog
- `ColorDialog` - Color dialog
- `DatePicker` - Date picker
- `TimePicker` - Time picker

Notifications
-------------

- `Toast` - Toast notification
- `ToolTip` - Tool tip
- `StatusTip` - Status tip

Theme
-----

- `Theme` - Theme mode enumeration
- `ThemeColor` - Theme color class
- `setTheme()` - Set theme mode
- `setThemeColor()` - Set theme color

Icons
-----

- `FluentIcon` - Built-in icon enumeration
